# AlmosaferTask

**Framework:**

1. Language: JAVA

3. Management & Execution: Maven & TestNG

**Pre-requisites (Installed & Configured):**
1. Java.
2. Maven

**Run Steps:**
1. Pull the project
3. Go to the projects path.
4. Pass below command:

- mvn clean test -P tests

MainProjectPath/src/Reports/Report.html
